# static-ffmpeg-build

A simple harness to build a custom ffmpeg useful to DCE for matterhorn
clusters.

## Getting Started

Run `./build.sh` on a debian jessie (and probably ubuntu 14.04) host. `ffmpeg`
will be statically compiled into `./bin`.
